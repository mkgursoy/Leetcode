# Leetcode Problems

[Leetcode](https://leetcode.com) is a site mostly made for programmers who are studying for job interviews, including problems in three categories being Easy, Medium and Hard. I mostly solve Hard problems here, and will add problems I have solved on this project with their solutions right under the answer code being in comments.

---


## Recommended Way Of Usage

This project includes both codes and solution explanations, which I have tried to make as easy to read and understand as possible. I encourage you not to use these solutions by simply copying them and submitting to Leetcode, instead use it as a learning opportunity. Especially because these are actual problems you can come across on job interviews and you should be ready once you are faced them.

Whenever you feel stuck on a problem, or simply feel like you can't solve it, you can read the solution of the problem, understand the idea, learn possible new information and solve the problem and many more in result.
